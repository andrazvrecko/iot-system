# Simulation of scalable IoT System

### DESCRIPTION

Scalable and highly available system. Can be scaled both vertically and horizontally.

Components:
  - Mosquitto Broker Cluster
  - MQTT-Kafka Bridge
  - Kafka Cluster
  - Kafka Streams
  - InfluxDB
  - Telegraf
  - Express Proxy
  - React Dashboard

### RUN
```
docker-compose up
```
